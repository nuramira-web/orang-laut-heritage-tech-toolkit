# Oral History Companion

The Oral History Companion is an AI-assisted tool to help interviewers ask richer, more careful follow-up questions during or after oral history interviews.

It is especially useful for younger volunteers, students, or researchers who may not yet know how to recognise culturally important openings in a conversation.

## It can help with

- follow-up questions
- missing context
- sensitive topic warnings
- suggested themes
- possible archive tags
- prompts for place, food, ritual, language, and maritime knowledge

## It should not

- interrupt the speaker
- force a topic
- impose interpretation
- treat AI suggestions as authoritative
- push sensitive memories
