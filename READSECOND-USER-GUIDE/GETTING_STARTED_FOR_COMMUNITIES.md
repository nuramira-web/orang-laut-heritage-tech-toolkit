
# Getting Started for Communities

The Pulau Brani Project welcomes contributions from:

* former island residents
* descendants
* families
* elders
* community members
* storytellers
* cooks
* fishers
* educators
* volunteers

---

# Ways to participate

You may contribute:

* oral histories
* photographs
* dialect words
* recipes
* maritime knowledge
* rituals and festival memories
* pronunciation recordings
* maps and routes
* family archives

---

# Important rights

Contributors may:

* update permissions
* request removal
* request anonymisation
* withdraw consent
* restrict AI use
* request community review

---

# Suggested first contribution

Start small.

Examples:

* record one disappearing phrase
* upload one photograph
* identify one remembered place
* document one recipe
* explain one maritime term
