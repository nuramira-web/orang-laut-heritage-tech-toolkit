# User Guide

## Orang Laut Heritage Tech Toolkit

## Pulau Brani Project

Welcome to the Orang Laut Heritage Tech Toolkit.

This project is an evolving, community-led cultural stewardship ecosystem exploring how archives, oral history, participatory storytelling, governance systems, education, and AI-assisted tools may support maritime heritage and intergenerational memory in increasingly digital futures.

The project focuses especially on:

* Pulau Brani
* Orang Laut communities
* Orang Pulau communities
* Singapore’s southern islands
* maritime memory
* oral heritage
* dialect and vocabulary
* foodways
* rituals and festivals
* migration and resettlement
* community storytelling

---

# Governance and stewardship first

The project approaches cultural memory as relational rather than purely informational.

This means:

* consent matters
* community review matters
* uncertainty matters
* cultural nuance matters
* contributor dignity matters
* AI outputs require review
* not everything should be public
* archives are not neutral

Please review:

* governance/
* LICENSES.md
* TK_LABELS_NOTICE.md
* AI_USE_POLICY.md
* ARCHIVE_ACCESS_POLICY.md

before contributing or reusing materials.

# Understanding project status labels

Some systems are conceptual while others are operational.

## Conceptual

Early-stage framework or idea.

## Prototype

Working experimental tool.

## In Development

Actively evolving system.

## Operational

Stable and usable system.

## Community Review Needed

Requires stewardship or contributor review.

---

# AI systems and important cautions

AI systems in this repository are experimental.

AI may:

* hallucinate
* flatten nuance
* overgeneralise
* mistranslate dialect
* remove uncertainty
* misread emotion
* simplify ritual meaning

AI outputs should never be treated as authoritative historical truth.

Human and community review remain essential.

# Long-term vision

The project explores how communities, institutions, educators, technologists, and younger generations may collaboratively steward memory, archives, language, and public storytelling in increasingly AI-mediated futures.

The goal is not only to preserve heritage.

The goal is to cultivate:

* stewardship
* participation
* literacy
* creativity
* ethical reflection
* intergenerational continuity
* responsible technological experimentation

around cultural memory.
