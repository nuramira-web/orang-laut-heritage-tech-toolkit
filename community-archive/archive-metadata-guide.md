# Archive Metadata Guide

Metadata helps future users understand what an archive item is, where it came from, and how it may be used.

## Required fields

- `archive_id`: Unique item ID, e.g. `PB-ARCH-001`
- `title`: Public-facing title
- `material_type`: Photograph, oral history, recipe, map, object, letter, audio, video, vocabulary, etc.
- `contributor_name`: Name of contributor
- `preferred_public_credit`: How the contributor wishes to be credited
- `community_or_island_connection`: Pulau Brani, Southern Islands, Orang Laut, Orang Pulau, etc.
- `date_or_period`: Date or approximate period connected to the item
- `location`: Relevant place
- `description`: Short description
- `story_context`: Longer explanation or memory
- `language_or_dialect`: Language or dialect used
- `public_use`: Yes / No / Limited
- `education_use`: Yes / No / Limited
- `ai_use`: Yes / No
- `sensitivity_level`: Public, community review needed, family restricted, culturally sensitive, spiritually sensitive
- `review_status`: Pending, approved, restricted, do not publish

## Optional fields

- `people_mentioned`
- `places_mentioned`
- `rituals_or_festivals`
- `foodways`
- `maritime_knowledge`
- `related_interview_id`
- `related_map_location_id`
- `file_name`
- `file_format`
- `date_added`
- `reviewed_by`
- `date_reviewed`
- `notes`
