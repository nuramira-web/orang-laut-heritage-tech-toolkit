# Interactive Memory Map

A simple map-based tool for documenting places connected to Orang Laut, Orang Pulau, Pulau Brani, and Singapore’s maritime communities.

Community members can use this to record:

- Former homes
- Jetties
- Mosques
- beaches
- fishing areas
- gathering places
- routes between islands
- places linked to rituals, food, work, or memory

Each map point may include a short description, image, audio clip, oral history reference, and consent status.
