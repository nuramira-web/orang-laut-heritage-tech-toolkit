# Ethical Persuasion Checker

The Ethical Persuasion Checker reviews advocacy, storytelling, presentation, and public communication materials for ethical risks.

It helps identify:

- sensationalism
- overstatement
- trauma aesthetics
- romanticisation
- cultural flattening
- unsupported claims
- extractive storytelling
- unclear consent
- weak community grounding

## Core principle

Good advocacy should persuade without distorting, exploiting, or oversimplifying community memory.
