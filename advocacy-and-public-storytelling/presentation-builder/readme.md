# Presentation Builder

The Presentation Builder helps students, educators, community members, and heritage practitioners structure thoughtful and ethical public presentations.

The tool supports:

- talks
- school presentations
- workshops
- public storytelling
- exhibition framing
- community briefings
- museum presentations
- advocacy presentations

The goal is not persuasive spectacle.

The goal is clear, emotionally intelligent, evidence-grounded communication.

---

# What the tool helps generate

- presentation arcs
- narrative structure
- slide flow
- audience framing
- emotional pacing
- speaking notes
- calls to action
- reflection prompts

---

# Core principle

A strong presentation should deepen understanding, not flatten complexity.
