# Memory Constellation Engine

The Memory Constellation Engine is an experimental AI-assisted heritage exploration system for the Pulau Brani Project.

Rather than treating archives as isolated objects, the engine connects memories, places, rituals, foodways, emotions, people, objects, dialect words, sea routes, photographs, and oral histories into navigable relationships.

The goal is to help users explore heritage through connection, not only keyword search.

## What it does

The engine can help users discover relationships between:

- oral histories
- archive objects
- recipes
- photographs
- rituals
- places
- emotions
- dialect words
- maritime knowledge
- family memories
- community practices

## Example

A user exploring “Malam Tujuh Likur” may discover related:

- Ramadan memories
- Hari Raya preparations
- family stories
- lamps and light
- food traditions
- women’s work
- prayer practices
- emotional themes such as anticipation, belonging, and loss

## Principles

1. Community stewardship first
2. Source-grounded interpretation
3. Consent-aware retrieval
4. No invention of history
5. Emotional nuance matters
6. AI supports discovery, not authority
7. Sensitive materials require restriction
8. Human review remains essential
