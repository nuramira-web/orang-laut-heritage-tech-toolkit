# Emotional Taxonomy

The Memory Constellation Engine recognises that heritage is not only factual. It is also emotional, relational, and embodied.

## Core emotional themes

- belonging
- loss
- longing
- joy
- pride
- grief
- tenderness
- resilience
- displacement
- continuity
- nostalgia
- reverence
- wonder
- humour
- intimacy
- care

## Use guidelines

Emotional tags should be used carefully.

They should be based on:

- the contributor’s own words
- oral history context
- family interpretation
- community review

AI may suggest emotional themes, but humans should review them.

Do not impose emotion where it is not evident.
