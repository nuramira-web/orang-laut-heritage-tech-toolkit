# AI Layer: Connection Suggester

This tool adds an AI-assisted layer to the Memory Constellation Engine.

It reads approved community archive items and suggests relationships between them, such as:

- shared themes
- related rituals
- connected places
- emotional patterns
- foodway links
- maritime knowledge links
- oral history connections

## What it produces

The script generates:

`ai-generated-memory-graph.json`

This file can be used by the constellation browser to display AI-suggested memory relationships.

## Important principles

AI-generated relationships are not final heritage interpretations.

They are draft suggestions that require:

- human review
- community review where appropriate
- source checking
- sensitivity review

## What the AI may do

The AI may suggest:

- possible links between archive items
- emotional themes
- relationship types
- search/discovery pathways
- interpretive clusters

## What the AI must not do

The AI must not:

- invent missing history
- create fake family relationships
- reveal restricted materials
- override community interpretation
- present uncertainty as fact
- flatten Orang Laut or Orang Pulau identities into one story
