# AI Story Weaver

The AI Story Weaver helps turn approved oral history fragments, photographs, recipes, objects, or memory notes into structured heritage story drafts.

It does not invent history. It helps contributors organise memory with care.

## It can help create

- short story summaries
- exhibition captions
- family memory notes
- educational text
- oral history excerpts
- website blurbs
- suggested follow-up questions

## Core rules

1. Use only approved source material
2. Do not invent missing details
3. Preserve the contributor’s voice
4. Keep uncertainty visible
5. Flag sensitive content
6. Require human review before publication
