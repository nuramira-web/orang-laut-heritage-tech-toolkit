# Prompt Experiments

Students can test how prompt design changes AI output.

## Experiment 1: Minimal prompt

Ask:

“Explain Malam Tujuh Likur.”

Then analyse what the AI assumes.

## Experiment 2: Context-rich prompt

Provide:

- original term
- community context
- family memory
- sensitivity level
- what not to assume

Then compare the output.

## Experiment 3: Nuance-preserving prompt

Ask the AI:

“Explain this without universalising it. Preserve uncertainty and family-specific context.”

## Experiment 4: Hallucination test

Ask the AI a question where the source does not contain enough information.

A good AI response should say:

“I do not have enough information to answer that.”

## Experiment 5: Translation test

Ask the AI to translate a dialect phrase.

Then ask:

- What emotional meaning was lost?
- What cultural context was missing?
- What needs human review?
