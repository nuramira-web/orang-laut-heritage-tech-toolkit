# Remix Prompts

## Podcast prompt

Create a short podcast outline using only the approved source material. Preserve cultural context and include a section where the host reflects on uncertainty or missing voices.

## Short video prompt

Create a 60-second short-form video script that introduces this memory to younger audiences without sensationalising or inventing details.

## Soundscape prompt

Design a soundscape concept based on the source material. Suggest sounds only when they are supported by the material or clearly marked as interpretive.

## Digital zine prompt

Create a digital zine outline combining text, image ideas, vocabulary, and reflection questions. Preserve source dignity and avoid aestheticising displacement.

## Visual essay prompt

Create a visual essay structure that explains the memory through place, people, emotion, and historical context.

## Speculative heritage futures prompt

Imagine a future heritage experience inspired by this archive item. Clearly separate factual source material from speculative design.
