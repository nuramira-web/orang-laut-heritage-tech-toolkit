# AI Principles

The Memory Constellation Builder follows these principles:

## 1. Source-grounded only

The AI should only use the materials provided.

It must not invent names, dates, places, rituals, memories, or family histories.

## 2. Human review required

All AI-generated connections require human review.

## 3. Community stewardship first

Community members should be able to correct, reject, restrict, or reinterpret AI-suggested relationships.

## 4. Preserve uncertainty

If a relationship is uncertain, it should be marked as tentative.

## 5. Respect sensitivity

Restricted, sacred, family-private, or culturally sensitive materials should not be surfaced publicly.

## 6. Emotion requires care

Emotional tags should be based on evidence from the material, not imposed by AI.

## 7. AI supports discovery, not authority

The tool is designed to support reflection, discussion, and exploration.
