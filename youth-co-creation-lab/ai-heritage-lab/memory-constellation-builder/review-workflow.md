# Review Workflow

AI-generated memory constellations require review before public use.

## Step 1: Generate draft

Run the Memory Constellation Builder on approved source materials.

## Step 2: Review nodes

Check:

- Are titles accurate?
- Are descriptions source-grounded?
- Are themes appropriate?
- Are emotional tags supported?
- Is anything sensitive?

## Step 3: Review links

For each suggested link, ask:

- Is this relationship supported by the source material?
- Is it interpretive or factual?
- Should it be public?
- Does it require community review?
- Does it risk oversimplifying the memory?

## Step 4: Mark status

Use:

- needs_review
- approved
- revise
- reject
- community_review_needed
- restricted

## Step 5: Publish only reviewed material

Only approved nodes and links should appear in public-facing tools.
