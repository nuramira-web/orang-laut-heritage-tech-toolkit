# orang-laut-heritage-tech-toolkit

An open-source toolkit to support Orang Laut, Orang Pulau, and maritime heritage communities in documenting, preserving, and sharing their histories using accessible digital tools.

This project begins from the Pulau Brani Project and is designed to support community-led documentation, oral history, digital storytelling, AI-assisted transcription and translation, and responsible public engagement.

## What this toolkit is for

This toolkit helps communities, researchers, educators, students, and heritage practitioners:

- Collect oral histories with care and consent
- Document rituals, festivals, foodways, maritime knowledge, and village life
- Use AI tools responsibly for transcription, translation, summary, and discovery
- Build simple public education resources
- Create future-ready digital archives and storytelling interfaces

## Guiding principles

1. Community first  
2. Consent before collection  
3. Cultural context matters  
4. AI should support, not replace, human memory  
5. Heritage should be accessible across generations  
6. Tools should be simple, low-cost, and reusable  

## Phase 1: Starter Toolkit

Phase 1 includes:

- Oral history interview guides
- Consent and ethics templates
- Metadata templates
- AI transcription and translation workflows
- Prompt libraries
- Storytelling templates
- Website copy for public engagement

## Future phases

Future phases may include:

- Interactive community maps
- Public memory submission forms
- AI-assisted archive search
- Responsible heritage chatbots
- Youth education resources
- Multilingual storytelling interfaces

## Who this is for

This toolkit is especially intended for Orang Laut, Orang Pulau, and island-descended communities, but may also be useful for other community heritage projects in Singapore, Southeast Asia, and maritime regions globally.

## Contributing

We welcome contributions from community members, researchers, technologists, educators, students, designers, and heritage practitioners.

All contributions should respect community consent, dignity, cultural nuance, and responsible use of technology.
